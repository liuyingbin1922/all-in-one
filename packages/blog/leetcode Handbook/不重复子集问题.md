# 求解不重复子集问题

## 背景

在数组这一专题中 ， 有一个使用回溯算法求解不重复子集问题。因为其使用的回溯算法是常见的解题思路，所以特此单独作为一个章节介绍该试题。


## 内容

### 回溯算法的核心知识点

回溯算法是过程其实可以总结为：

> 路径：选择决策树的路径

> 选择列表： 选择的数组

> 结束条件： 到达决策树的底层，无法再次进行决策；


- 代码模版：

```js

const result = []
function backtrack(路径, 选择列表):
    if (满足结束条件)
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```
